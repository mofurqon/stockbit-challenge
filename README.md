### Question 2 Express.js
make sure you have installed node and run this command
```bash
cd question2
```
install the dependency
```bash
npm install
```

run the server

```bash
npm run start
```
open `http://localhost:8080` in the browser 

unit testing
```bash
npm run test
```

live server `https://omdb-stockbit.herokuapp.com`
